# EShop

Модуль интернет-магазина для платформы Абрикос.


## Элементы шаблона

### product_list.html

Формирует список товара используя product_list_item.html

Параметры:

- forcontent
- limit
- notchildlist
- classcolumn

### product_list_item.html

Шаблон товара для списка.

Параметры:

- imgw, imgh - размер картинки
